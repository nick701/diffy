# Corresponding Source Code

Before publishing a Diffy binary release, publish its corresponding source at
https://github.com/nick701/diffy using a Git tag matching the app version
(for example, `v0.6.0`) or attach the matching source archive to the release.

Build instructions and the exact Sparkle dependency revision are in the
matching source tree's `README.md`, `Package.swift`, and `Package.resolved`.
